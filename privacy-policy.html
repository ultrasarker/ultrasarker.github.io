Privacy Policy

Introduction

As the developer of Processor, I am committed to protecting the privacy and data of all users who interact with my Discord bot. This privacy policy outlines how information is collected, used, and protected when you use Processor.

Data Collection

Processor collects and stores the following information:

  • Discord User IDs (for user-specific functionality and settings)
  • Server IDs (for server-specific configurations)
  • Command usage data (for improving bot functionality)

Data Usage

The collected data is used exclusively for:

  • Providing and improving bot functionality
  • Maintaining user preferences and settings
  • Analyzing command usage to enhance user experience
  • Debugging and resolving technical issues

Data Protection

I implement several measures to protect your data:

  • All data is stored securely with industry-standard encryption
  • Access to data is strictly limited to essential bot operations
  • Regular security audits and updates are performed
  • No personal data is shared with third parties

User Rights

Users have the right to:

  • Request a copy of their stored data
  • Request deletion of their data
  • Opt-out of non-essential data collection
  • Report any privacy concerns or incidents

Ethical Commitments

As the developer of Processor, I commit to:

  • Transparency in data collection and usage
  • Minimizing data collection to only what's necessary
  • Regular updates about privacy policy changes
  • Prompt response to privacy concerns and questions
  • Ensuring the bot's features respect user privacy
